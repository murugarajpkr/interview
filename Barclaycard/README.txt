*****************
Release Notes
*****************
1. Barclaycard Project has the implementation of solution for "Airport Baggage" Problem
2. The Barclaycard Project is implemented as Maven Project.
3. com.interview.exam.solution.AirportBaggageImpl Class has the implementation of "Airport Baggage" Problem.
4. Created com.interview.exam.solution.AirportBaggageImplTest Junit Class as Unit Testing Class for invoking the implementation Method.


************************************************
Steps to Execute the "Airport Baggage" Solution:
************************************************
1. Import the Barclaycard project as "Maven" project in the IDE(example: Eclipse, IntelliJ IDE).
2. Perform "Clean" and "Compile" of the project.
3. Navigate to the com.interview.exam.solution.AirportBaggageImplTest Junit Class.
4. Run AirportBaggageImplTest (by right clicking on this Junit Class).
5. The Input and the Output Result will be printed on the Console.
